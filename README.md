# qiita-organizations-graph

Qiita organizations graph

## how to web scraping

execute command below:

```bash
$ npm install
$ npm start
```

install dependency modules, and execute web scraping.

## how to preview page

requirements:

- Ruby
- RubyGems
- Rake
- Bundler

```sh
$ rake install 
$ rake
```
